# lucid-auth

MQTT credential and ACL management service for LUCID.

This service talks to the EMQX management API. It does not authenticate MQTT
clients itself. Instead, it provisions broker-native resources so EMQX can do:

- password authentication from its built-in database for local users
- LDAP authentication as the next fallback authenticator
- built-in database authorization rules for per-user ACLs

## Roles

### Agents

- Auth: EMQX built-in database
- Username: `{agent_id}`
- Password: generated by `lucid-auth`
- Agent ID format: lowercase letters, numbers, and `_` only (`[a-z0-9_]+`) to match `lucid-agent-core`
- ACL: publish only to own `lucid/agents/{agent_id}/...` topics and subscribe only to own `cmd/#` topics

### Central Command

- Auth: EMQX built-in database
- Username: `BOOTSTRAP_CC_USER` or `central-command`
- Password: generated by `lucid-auth` or bootstrapped from env
- ACL: publish all command topics, subscribe all agent/component data topics, access researcher topics

### Researchers

- Auth: LDAP directly in EMQX
- Password: not stored here
- ACL: built-in database authorization rules scoped to `lucid/researcher/{username}/#`

## Environment

| Variable | Description |
|---|---|
| `EMQX_URL` | EMQX dashboard / management API base URL |
| `EMQX_USERNAME` | EMQX dashboard username |
| `EMQX_PASSWORD` | EMQX dashboard password |
| `BOOTSTRAP_CC_USER` | Optional built-in MQTT username to create on startup |
| `BOOTSTRAP_CC_PASSWORD` | Optional password for the bootstrap central-command user |

## CLI

```bash
python manage.py add-agent robot_01
python manage.py list-agents
python manage.py add-cc
python manage.py add-user forfaly
python manage.py list-users
```

## API

| Method | Path | Purpose |
|---|---|---|
| `GET` | `/health` | Liveness |
| `POST` | `/agents/{agent_id}` | Provision local agent password in EMQX |
| `DELETE` | `/agents/{agent_id}` | Revoke agent |
| `GET` | `/agents` | List built-in database agents |
| `POST` | `/cc` | Provision local central-command password in EMQX |
| `DELETE` | `/cc` | Revoke central-command |
| `POST` | `/users/{username}` | Provision LDAP-backed researcher ACL |
| `DELETE` | `/users/{username}` | Revoke researcher ACL |
| `GET` | `/users` | List researcher ACL entries |

## Local run

```bash
pip install -r requirements.txt
uvicorn api:app --reload --port 4000
```
